Project Overview

The To-Do List is a simple web application that helps users manage their daily tasks efficiently. Users can add, update, delete, and mark tasks as completed, providing a simple yet effective way to organize their day.

This project is designed to improve productivity and task management while demonstrating core CRUD functionality for web development learning purposes.

Features

Add new tasks with titles and descriptions

Mark tasks as completed or pending

Edit and delete existing tasks

Responsive UI for desktop and mobile

Simple and intuitive design

Technologies Used

Frontend: HTML, CSS, JavaScript


Tools: VS Code, Git
